# [세계 1등 우량주 투자 매뉴얼]
- 포지션 판단의 기준은 종가를 기준으로 함
- 나스닥 -3% 이상 하락이 발생하지 않은 평소에는 리밸런싱만 진행
- 나스닥 -3% 이상 하락 ] 발생하면 이 시기를 ‘위기’로 정의
- 나스닥 -3% 이상 하락 ] 4회 이상 발생하면 ‘공황’으로 정의

---

## 1. 평상시 [신규 매수 진입]

* **1등과 2등의 격차가 10% 이상 [시가총액 기준]**
    * 1등주 매수
        * 팬덤이 있는 / 천재 CEO가 있는 회사가 있다면 전체 자산의 10% 투자하기
        * 없다면 기존의 방식대로 100% 1등주 투자
* **1등 시가총액 기준 10% 이내인 종목이 있다면**
    * 1등주와 같은 비율로 매수 (1등주 : 2등주 : 3등주 = 1:1:1)
        * 매수는 항상 종가에 매수

---

## 2. 평상시 [1등주 매수 상태]

* **2등주가 1등주를 추월하면**
    * 1등주 : 2등주 = 1:1
* **1등과 격차가 10% 이내에 있다가 2등주 격차가 15% 밖으로 벌어지는 경우**
    * 2등주 매도 / 1등주 매수
        * 매도는 시가에 매도
* **-3%도 안떴는데 일등주 전고점 대비 하락시**
    * 25% 현금화 리밸런싱 전략 [-2.5% 하락마다 10%씩 현금화]
        * -2.5% 하락마다 : 1등주 10% 매도 후
        * 개별 종목 이슈시 : QQQ 10% 매수 -> 반등하면 반대로 리밸런싱
        * 전체 시장 위기시 : 현금 보유 -> 나스닥 -3% 발생하면 : 비율에 맞춰 말뚝박기 진행
        * 마삼 없이 리밸런싱 기간 중 최종 리밸런싱 구간에서 2구간 상승하면 주식 전량 매수 [1구간 기준 : 양적완화 기간 X - 5.0% / 양적완화 기간 - 2.5%]
* **보유 종목 어닝 발표시**
    * **불노 하락** : 어닝 1주일 전, 2주일 전 모두 상승 그러나 어닝쇼크 발생
        * 어닝일 종가에 [불노 하락 확정시] 종목 매도
            * 일주일 후 재매수
    * **단순 어닝쇼크** : 어닝 1주일 전, 2주일 전 모두 하락 또는 둘 중 하나 하락 / 그리고 어닝쇼크 발생
        * 계속 보유

## 3. -3% 발생 : 위기 발생 [나스닥 지수 기준]

* **-3%가 뜨면**
    * **고점대비 하락 비율에 따라 말뚝 유지**
        * -3% 발생시 +수익 중이라면 -> 말뚝 비중을 제외하고 매도
        * -3% 발생시 -손실 중이라면 -> 전량 매도 후 말뚝 재진입
    * **제로금리 기간이 아닐 때 [=기준금리 0.25% 이상] -> 50% 말뚝박기 진행**
        * 말뚝은 아래로 내려갈수록 기존 말뚝은 뽑아서 다시 박는다 [워시세일 손절 절세팁] -> 당해 연도 이익이 250만 원을 초과할 경우만
        * * -5% 하락할 때마다 [주식 비율 10%]
        * * 양적완화 전 -> 현금 비중으로 TLT 10% 매수
    * **제로금리 기간일 때 [=기준금리 0~0.25%] -> 25% 말뚝박기 진행**
        * 말뚝은 아래로 내려갈수록 기존 말뚝은 뽑아서 다시 박는다 [워시세일 손절 절세팁] -> 당해 연도 이익이 250만 원을 초과할 경우만
        * * -2.5% 하락할 때마다 [주식 비율 10%]
        * * 제로금리 기간 -> 현금 비중으로 IAU 10% 매수
        * * 금선물 증거금 제재 중일 경우에는 -> 현금 10% 보유
    * **말뚝박기 기간에 오름세로 돌아섰을 경우**
        * 시장가가 평단가보다 낮은 가격이라면 과감하게 확보한 현금을 투입하자! [평단가를 낮춰 시장수익률을 이기는 것이 목표]
    * **양적완화 이전**
        * 주식 매도 물량 -> TLT 매수
        * 미국 연준(Fed)의 금리인하 (3, 6, 9, 12월) 또는 양적완화 이후에는 -> TLT 매도 / IAU or 현금(달러) 보유
    * **제로금리 / 양적완화 시작 [단기금리가 0~0.25% 구간일 때]**
        * IAU 또는 GLD(금연동ETF) / TIP(물가연동채권) 매수 [1:1 비율로]
        * 금리인상 또는 양적완화 축소시 or 공황이 끝나면 -> IAU, GLD / TIP 전량 매도 / 현금(달러) 보유
    * **경기 호황으로 금리가 높은 상태 or 경기 침체로 금리가 낮을 경우**
        * * 미국채 수익률이 내려가는 추세라면 -> TLT로 헷지
        * * 미국채 수익률이 올라가는 추세라면 -> 달러로 헷지
        * * 방향성이 뚜렷하지 않은 경우라면 -> 달러로 헷지

* **마지막 -3%가 뜬 날로부터 1개월 +1일이 되는 날**
    * TLT 매도 / 1등주 매수 -> 위기가 끝나는 날로 지정 종가에 매수

* **같은 달에 -3%가 4번 이상 뜨면**
    * 공황 확정 -> 아래의 공황 매뉴얼을 따른다

## 4. -3% 4번 이상 발생 [공황 확정]

* **TLT를 들고가는 경우**
    * 공황 초반에만 써먹어야 한다 [급하게 오르고 급하게 떨어진다]
        * 미국 연준(Fed)의 금리인하 / 양적완화 조치 -> TLT 매도 / 현금(달러) 보유
        * 매수세력의 차익 실현
        * 중앙은행들의 미국채 매도
* **IAU, GLD / TIP를 들고가는 경우**
    * 양적완화시에만 써먹어야 한다
        * 미국 연준(Fed)의 금리인상 또는 양적완화 축소시에는 -> IAU, GLD / TIP 전량 매도 / 현금(달러) 보유
* **공황으로 인해 환율이 급등하면 [원화 가치 하락 / 달러 가치 강세]**
    * 달러 -> 원화
        * 이머징 마켓 환율 안정화 [원화 가치 상승 / 달러 가치 하락] -> 원화 -> 달러
