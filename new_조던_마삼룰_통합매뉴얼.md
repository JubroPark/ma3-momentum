## 0. 전제 조건 · 용어 정의

|------|------|

| **지수 기준** | 마삼 판정은 **나스닥 종합지수(IXIC)** 일간 등락률 |
| **매수 시점** | 원칙적으로 **종가 매수** (일부 매도는 시가) |
| **대상 종목** | 전 세계 시가총액 **1등주**(기업 분석 불필요, 가격이 가치를 반영한다는 전제) |
| **운용 사이클** | 평소엔 리밸런싱만 → 마삼 발생 시 위기 대응 → 안정 시 재진입 |

### 0-2. 핵심 용어

| 용어 | 정의 |
|------|------|
| **마삼(-3%)** | 나스닥 종합지수가 종가 기준 전일 대비 **-3% 이상 하락**한 날 |
| **위기** | 마삼이 1회라도 발생한 국면. 종료 = 마지막 마삼일 **+ 1개월 + 1일**(달력 기준) 동안 추가 마삼 無 |
| **공황** | **같은 달력 월(月) 안에 마삼 4회 이상** 발생한 국면. 종료 = 마지막 마삼일 **+ 2개월 + 1일** 동안 추가 마삼 無 |
| **마삼 카운팅** | **달력 월 기준**으로 집계, 월이 바뀌면 카운트 리셋 (※ 30일 롤링 아님) |
| **말뚝박기** | 하락 구간마다 현금을 분할 투입해 평단가를 낮추는 분할 매수 전략 |
| **올인(몰빵)** | 보유 현금을 전량 투입하는 행동 |
| **구간** | 가격 변동을 일정 % 단위로 나눈 그리드. **제로금리 = 2.5% / 비제로금리 = 5%** |
| **V자 반등** | 바닥에서 구간 단위로 빠르게 되돌리는 반등 (올인 트리거로 사용) |
| **제로금리** | 미 기준금리 0~0.25% 구간 |
| **비제로금리** | 미 기준금리 0.25% 초과 |

### 0-3. 핵심 철학 (왜 이렇게 하는가)

1. **잃지 않는 것이 최우선.** 큰 하락장의 초입(-3%)에서 일단 빠져 계좌를 방어한다.
2. **감정을 배제한 규칙 기반 매매.** 숫자와 조건으로만 판단해 충동·공포·탐욕을 차단한다.
3. **시총 1위만 산다.** 개별 기업 분석 부담 없이, 시장이 인정한 1등에 올라탄다.
4. **수익 극대화보다 멘탈·원금 보존.** 상승장 일부를 포기하는 대신 공황을 회피한다.

### 0-4. 위기 / 공황 기간 판정 규칙 ★중요

> **마삼 카운팅과 기간 산정은 "달력(캘린더)" 기준**이다. "발생일로부터 며칠" 식의 30일 롤링이 아니다.

**(1) 마삼 카운팅 = 달력 월 기준 (월이 바뀌면 리셋)**

- 같은 달력 월 안에서 마삼 횟수를 센다. 월이 넘어가면 카운트는 0으로 리셋.
- 예시: `5/20, 5/21, 5/28, 6/7`에 마삼 발생 → **5월 3회**(위기), **6월 1회**(위기). → "한 달 4회"가 아니므로 **공황 아님**.

**(2) 위기 종료 = 마지막 마삼일 + 1개월 + 1일 (달력)**

- "익월(다음 달) 동일 일자 + 1일"까지 추가 마삼이 없으면 위기 종료.
- 예시: `6/17` 마삼 1회 → **7/18**까지 추가 마삼 없으면 위기 종료.
- 대기 기간 중 새 마삼이 뜨면 그날을 새 기준일로 타이머 재산정.

**(3) 공황 = 같은 달력 월에 마삼 4회 이상**

- 공황 종료 = 마지막 마삼일 + **2개월 + 1일**(달력) 동안 추가 마삼 없으면 종료.

> 📌 월말 엣지 케이스: 익월에 동일 일자가 없으면(예: 1/31 → 2/31 없음) 해당 월 말일 +1일 등 달력 관례로 처리. (사전에 규칙 고정 권장)

## 1. 평상시 — 종목 선정 & 신규 진입

> 마삼이 발생하지 않은 평소에는 아래 종목 규칙 + 리밸런싱(2장)만 운용한다.

### 1-1. 1등 vs 2등 시총 격차로 비중 결정 (시가총액 기준)

| 상황 | 행동 |
|------|------|
| **1등·2등 격차 10% 초과** | 1등주에 집중 (원칙적으로 100% 1등주) |
| **1등·2등 격차 10% 이내** | 1등 : 2등 (: 3등) = **1 : 1 (: 1)** 동일 비중 매수 |

- 매수는 항상 **종가 매수**.

### 1-2. "1등주" 정의

- **1등주 = 그 시점의 전 세계 시가총액 1위 기업.** 특정 티커로 고정하지 않는다.
- 시총 1위는 언제든 바뀌므로 **운용 시 매번 시총 1위를 재확인**해 적용한다.
- (사우디 아람코처럼 사실상 거래·접근이 어려운 경우는 제외하고, 투자 가능한 글로벌 시총 1위를 기준으로 함.)

## 2. 평상시 — 보유 중 관리 & 현금화 리밸런싱

### 2-1. 1·2등 역전 / 격차 변화 대응

| 상황 | 행동 |
|------|------|
| 2등주가 1등주를 추월 | 1등 : 2등 = **1 : 1** 로 조정 |
| 격차 10% 이내였다가 **2등이 15% 밖으로 밀려남** | **2등주 매도 → 1등주 매수** (이때 매도는 **시가 매도**) |

### 2-2. 마삼 없이 1등주가 전고점 대비 하락할 때 — 25% 현금화 리밸런싱

> 마삼(-3%)이 안 떴는데도 1등주가 전고점 대비 흘러내리는 경우의 방어 전략.
> **1구간(=양적완화 기간 -2.5% / 그 외 -5%) 하락마다 10%씩 현금화.** 최대 25%까지.

| 트리거 | 행동 |
|--------|------|
| 1구간 하락마다 | **1등주 10% 매도** |
| 개별 종목 이슈일 때 | 매도분으로 **QQQ 10% 매수** → 반등 시 역리밸런싱(QQQ→1등주) |
| 전체 시장 위기 조짐일 때 | **현금 보유** → 나스닥 -3% 뜨면 비율 맞춰 말뚝박기로 전환 |
| 리밸런싱 막바지에 **2구간 상승**(되돌림) | **주식 전량 매수**(복귀) — 1구간 기준: 양적완화기 -2.5% / 그 외 -5% |

### 2-3. 과열 시 선제적 현금 확보

- 주가가 이평선을 크게 상회한다고 판단될 때(= "수익 자랑하고 싶을 때"): 보유 자산의 **20%를 미리 매도해 현금 확보**.
- 단, **현금화 리밸런싱이 우선**이므로 상황에 따라 보류 가능.
- 추가 자금 투입은 **RSI + MFI 두 지표가 모두 50 이하**일 때만. (2월·9월은 역사적 조정의 달)

## 3. 마삼(-3%) 발생 — 위기 대응 매뉴얼

> 나스닥 종가 -3% 확인 → 즉시 아래 절차. 핵심은 **일단 빠지고(또는 말뚝만 남기고), 현금으로 분할 재진입**.

### 3-1. -3% 뜬 날, 즉시 행동 (현 손익 상태에 따라)

| 현재 상태 | 행동 |
|-----------|------|
| **+수익 중** | **말뚝 비중을 제외**하고 매도 (수익 난 본 포지션은 정리, 말뚝은 유지) |
| **-손실 중** | **전량 매도** 후 → 말뚝박기로 **재진입** |

> 즉, 고점 대비 하락 비율에 따라 "말뚝을 유지/재진입"하는 구조.

### 3-2. 금리 환경별 말뚝박기 — 비중 조절 알고리즘 (핵심)

> **그리드 통일 원칙**: 제로금리 = **2.5%** 단위, 비제로금리 = **5%** 단위.
> 하락 1구간마다 주식 10% + 헤지자산 10%를 투입한다.

| 금리 환경 | 말뚝 목표 비중 | 하락 그리드 | 1구간 하락 시 행동 |
|-----------|---------------|------------|-------------------|
| **비제로금리** (기준금리 > 0.25%) | **~50% 말뚝** | **-5%** 마다 | 주식(1등주) 10% 매수 + (양적완화 전) **TLT 10% 매수** |
| **제로금리** (0~0.25%) | **~25% 말뚝** | **-2.5%** 마다 | 주식(1등주) 10% 매수 + **IAU 10% 매수** |

- **제로금리에서 금선물 증거금 제재 중**이면 → IAU 대신 **현금 10% 보유**.
- **말뚝 워시세일 절세팁**: 더 내려가면 기존 말뚝을 뽑아(매도) 다시 박는다(저점 재매수). 단, **당해 연도 실현이익이 250만 원을 초과할 경우에만** 손절 처리해 양도세 절감.
- **반등 전환 시**: 시장가가 평단가보다 낮으면, 확보한 현금을 **과감히 투입**(평단 낮춰 시장수익률 추월이 목표).

> 📌 **간이 버전(손닥터식 예시)**: "예상 최대 낙폭 -25%"로 잡으면 → 나스닥 -5%에 10%, -10%에 +10%(누적 20%)… -25%에 누적 50% 매수. 개인 성향에 따라 그리드·총량 조절 가능. (위 표가 더 정밀한 원본 기준)

### 3-3. 헤지(대피) 자산 운용

> 매도로 마련한 달러를 그냥 들고 있어도 되지만, 국면에 맞는 안전자산으로 추가 수익/방어를 노린다.

| 국면 | 보유 자산 | 청산/전환 시점 |
|------|----------|----------------|
| **양적완화 이전** | 주식 매도분 → **TLT 매수** | Fed 금리인하(3·6·9·12월) 또는 양적완화 시작 → **TLT 매도 / IAU·현금(달러)** 보유 |
| **제로금리 / 양적완화 시작** | **IAU(또는 GLD) : TIP = 1 : 1** | 금리인상·양적완화 축소 or 공황 종료 → **전량 매도 / 현금(달러)** |
| 미국채 수익률 **하락 추세** | **TLT로 헤지** | 추세 전환 시 |
| 미국채 수익률 **상승 추세** | **달러로 헤지** | — |
| 방향성 **불분명** | **달러로 헤지** | — |

| 티커 | 자산 |
|------|------|
| TLT | 미국 20년 국채 ETF |
| TIP | 미국 물가연동국채 ETF |
| IAU / GLD | 금 ETF |
| QQQ | 나스닥100 ETF (분산·갈아타기용, 7장) |

### 3-4. 환율 대응 (공황 시 변동성 확대 국면)

| 상황 | 행동 |
|------|------|
| 공황으로 환율 급등 (원화↓ / 달러↑) | 일부 **달러 → 원화** |
| 이머징 환율 안정화 (원화↑ / 달러↓) | **원화 → 달러** 복귀 |

### 3-5. 위기 종료 — 시간 기반 재매수

- **마지막 -3%가 뜬 날 + 1개월 + 1일**(달력 기준, 익월 동일 일자 +1일)이 "위기 종료일".
- 예: 6/17 마삼 → 7/18까지 추가 마삼 없으면 종료. 그날 **TLT(헤지) 매도 / 1등주 매수**(종가).
- 대기 기간 중 마삼 재발생 시 그날을 새 기준일로 타이머 리셋.
- 단, 그 전에 8거래일 연속 상승·전고점 돌파·V자 2구간(6장) 등 빠른 트리거가 뜨면 그걸로 조기 종료 가능.

### 3-6. 공황 전환 판정

- **같은 달력 월 안에 -3%가 4번 이상** → **공황 확정** → 4장 비상 매뉴얼로 전환.
- (월이 바뀌면 카운트 리셋. 0-4 참조: 5월 3회 + 6월 1회 = 공황 아님.)

## 4. 공황(마삼 4회+) — 비상 매뉴얼

> 공황은 "큰 폭등 뒤에 오는 진짜 하락장". 이때는 **욕심을 버리고 현금 보존을 최우선**으로 한다.

### 4-1. 공황 진입 조건

| 조건 | 비고 |
|------|------|
| **같은 달력 월 안에 마삼 4회 이상** | 핵심·유일 트리거. 월이 바뀌면 카운트 리셋(0-4 참조) |

> ※ 이전 정리본(B)의 "전년도 +45% 폭등" 전제는 **제외**. 공황 판정은 오직 "한 달(달력 월) 4회"로 확정.

### 4-2. 평시 위기 vs 공황 — 핵심 차이

| 항목 | 평시 위기 매뉴얼 | 공황 비상 매뉴얼 |
|------|------------------|------------------|
| 말뚝박기 | 진행 | 진행하다 **마삼 4회 누적 시 즉시 중단** |
| 중단 후 | — | **보유 주식 전량 매도 → 현금 100% 대기** |
| 고점 -30%/-15% 긴급 올인 | 있음 (정식 룰, 6-3) | 있음 — 현금 대기 중 도달 시 재진입 올인 |
| 올인 이후 운용 | 마삼 종료 → 리밸런싱 복귀 | **최고점 경신 전까지 리밸런싱·마삼 말뚝박기 모두 중단(완전 홀드)** (6-4) |
| 재매수 대기 | 마지막 -3%로부터 **1개월 +1일** | **2개월 +1일** (또는 조기 트리거) |
| V자 올인 트리거 | **2구간**(+5%/+10%) | **2구간**(+5%/+10%) — 동일 |

### 4-3. 공황 시 헤지 자산 주의

| 보유 자산 | 써먹는 국면 | 청산 트리거 |
|-----------|-------------|-------------|
| **TLT** | 공황 **초반에만** (급등·급락) | Fed 금리인하·양적완화 / 매수세력 차익실현 / 중앙은행 미국채 매도 → **TLT 매도, 현금(달러)** |
| **IAU·GLD / TIP** | **양적완화 시기에만** | 금리인상·양적완화 축소 → **전량 매도, 현금(달러)** |

### 4-4. 공황 흐름 요약

```
공황 진입(달력 월 마삼 4회)
   → 말뚝박기 즉시 중단 → 전량 매도 → 현금 100% 대기
   → 재진입 트리거 충족 시 올인:
        · 고점 대비 6구간 하락(비제로 -30% / 제로 -15%) → 긴급 올인
        · 2구간 반등(비제로 +10% / 제로 +5%) → V자 올인
        · 8거래일 상승 / 전고점 돌파 / 2개월+1일 경과
   → 올인 후에는 [최고점 경신 전까지] 리밸런싱·마삼 말뚝박기 모두 중단(완전 홀드)
   → 최고점 경신 후 평상시 모드 복귀
```

> 📎 역사적 참고: 나스닥은 큰 하락 후 **평균 ~30% 선에서 반등**. 40% 이상 하락은 단 2회 — 닷컴버블 **-78%**(고점 회복 ~15년), 리먼 **-55%**(~5년). 2022년 약 -35%.

## 5. 재진입 · 올인(몰빵) 조건

> 아래 트리거 중 **하나라도 충족되면 올인** → 마삼 종료 → 평상시 리밸런싱(2장)으로 복귀.
> (단 V자 2구간 올인은 별도 규칙 적용, 6장)

### 5-1. 올인 트리거 (평시 기준)

| # | 트리거 | 성격 / 신뢰도 |
|---|--------|---------------|
| ① | 마삼 후 **1개월 +1일** 추가 마삼 없음 | 시간 기반 (안정 확인, 추세 미확인) — *약* |
| ② | 나스닥 **8거래일 연속 상승** | 추세 전환 패턴 — *중* |
| ③ | **1등주 전고점 돌파** | 데드캣 초과 = 진짜 반등 — *강* |
| ④ | **나스닥 전고점 돌파** | 완전 회복 — *강* |
| ⑤ | 바닥 대비 **2구간 상승 (V자)** | 구조 기반 → 6장 별도 규칙 — *중* |

> ※ 신뢰도(약/중/강)는 참고 표시일 뿐. 원본 규칙은 "단일 신호 충족 = 올인". (등급 활용은 부록 Z 참조)

### 5-2. 공황 시 재진입 — 안전 진입

- **마지막 -3%로부터 2개월 +1일** = 공황 종료 확정일, 종가 매수.
- 또는 조기 진입: **8거래일 연속 상승** / **나스닥 전고점 돌파**(공황의 끝은 아니지만 진입 가능).

### 5-3. 재진입 시 개별 종목 분할 매수법

> 트리거 확정일(8거래일째 / 전고점 돌파일 / 2개월+1일 등)의 **종가 매수**를 기본으로, 개별 종목 당일 등락에 따라 분할:

| 개별 종목 당일 상태 | 분할 방식 |
|---------------------|-----------|
| **-1% ~ -2% 하락** | 35% / 35% / 30% — **하락일 3회에 나눠** 종가 매수 |
| **-2% 이상 폭락** | 35% / 35% / 30% — **당일 3분할** 종가 매수 |

### 5-4. 추가 자금 투입 규칙

- 여유 자금은 모아뒀다가 **RSI + MFI 모두 50 이하**일 때만 투입.
- 2월·9월은 역사적 조정의 달 — 자금 투입 기회로 인지.

## 6. V자 반등 — 세부 규칙

### 6-1. 구간(그리드) 정의

| 금리 환경 | 1구간 | **2구간 = 올인 트리거(평시)** |
|-----------|-------|------------------------------|
| 금리 환경 | 1구간 | **2구간 = V자 올인 트리거** |
|-----------|-------|------------------------------|
| 제로금리 | +2.5% | **+5%** |
| 비제로금리 | +5% | **+10%** |

> 구간 폭은 말뚝박기 하락 그리드(3-2)와 동일 단위 → 하락·상승을 같은 자(尺)로 측정.
> 공황 전용 "6구간 반등" 트리거는 두지 않는다(별도 규칙 아님).

### 6-2. V자 올인 후 운용 — 평시

```
바닥 확인 → 2구간 상승 감지 → 남은 현금 전량 올인
        ↓
   올인 후 (위기 종료까지) 리밸런싱 없이 말뚝박기 유지
        ↓
   다시 3구간 이하로 재하락? → 말뚝박기만 진행 (추가 리밸런싱 없음)
        ↓
   V자 올인 후 다시 3구간 하락 확정 시
        → V자 매수 물량 전량 매도 → 말뚝박기 재진입
```

### 6-3. 긴급 올인 (정식 룰) — 금리 환경별

> 잔여 현금을 전량 투입하는 두 갈래 중 **하락 경로**. 고점에서 **6구간(하락)** 깊이까지 떨어지면 긴급 올인.

| 금리 환경 | 1구간 | 긴급 올인 트리거 (고점 대비) |
|-----------|-------|------------------------------|
| 비제로금리 | -5% | **-30%** (= 6구간) |
| 제로금리 | -2.5% | **-15%** (= 6구간) |

- 평시·공황 공통으로 적용하는 정식 룰.
- 공황 국면에서는 현금 100% 대기 중 위 깊이 도달 시 이 규칙으로 재진입.

> **긴급 올인 vs V자 올인 (둘 중 먼저 충족되는 쪽)**
> - **하락 경로**: 고점 대비 6구간(-30% / -15%)까지 더 떨어짐 → **긴급 올인**
> - **상승 경로**: 거기까지 안 가고 저점에서 **2구간 반등**(+10% / +5%) → **V자 반등 올인** (6-1)
> 즉, 더 깊이 떨어지면 -30%/-15%에서 긴급 올인, 먼저 반등하면 2구간에서 V자 올인.

### 6-4. 공황 후 올인 처리 — 최고점까지 완전 홀드 ★

> **공황 확정 후** V자 반등 올인 **또는** 긴급 올인(-30%/-15%)을 실행했다면:
> - **최고점(전고점) 경신 전까지** 리밸런싱을 하지 않는다.
> - **마삼(-3%) 말뚝박기도 하지 않는다.**
> - 즉, 올인한 포지션을 **그대로 들고 최고점 경신까지 완전 홀드**한다.
> - 최고점을 경신하면 비로소 평상시 모드(1~2장)로 복귀.

> 📌 평시 V자 올인(6-2, "말뚝박기 유지")과 **다른 점**: 공황 후 올인은 말뚝박기조차 하지 않고 그냥 홀드.

## 7. QQQ 갈아타기 / 분산 매뉴얼

> ⚠️ 용어 주의: QQQ는 진짜 "헤지"가 아니라 **분산·베타 감소**(시장 동반 하락). 명칭은 "갈아타기/분산"으로 이해.

### 7-1. 발동 조건

- **1등주가 나스닥 마삼 없이 단독으로** 크게 빠질 때(어닝쇼크·섹터 이슈 등),
- **그리고 1등-2등 시총 차이가 10% 초과**라 종목 교체가 불가할 때.

### 7-2. 단계별 실행 (20% 단위)

| 1등주 하락폭 | 1등주 매도 | QQQ 비중 |
|------------|-----------|---------|
| -10% | 20% 매도 | 20% |
| -20% | +20% | 40% |
| -30% | +20% | 60% |
| -40% | +20% | 80% |
| -50% | 잔여 전량 | 100% |

### 7-3. 회복 시 역방향 전환

- **1등주 전고점 돌파** 또는 **MDD 회복 +10% 이상** 시 → QQQ를 **20%씩 매도**하며 1등주로 순차 복귀.

| 케이스 | 처리 |

| **월이 바뀜** | 마삼 카운트 0으로 리셋 (공황 판정은 달력 월 4회 기준) |
| 말뚝박기 중 **달력 월 마삼 4회 누적** | (공황) 즉시 중단 → 전량 매도 → 현금 대기 |

| 2등이 **15% 밖으로** 이탈 | 2등 시가 매도 → 1등 매수 |
| 평시 V자 올인 후 **3구간 재하락** | V자 물량 전량 매도 → 말뚝 재진입 |
| **공황 후** V자/긴급 올인 실행 | 최고점 경신까지 리밸런싱·말뚝박기 모두 중단(완전 홀드, 6-4) |
| 월말 일자 부재(예: 1/31+1개월) | 달력 관례로 처리(말일+1일 등), 사전 규칙 고정 |

## 9. 의사결정 플로우 (전체 흐름)

```
[매일 종가 확인]
        │
   나스닥 -3% ?  ──No──► 평상시 모드
        │                 ├─ 1·2등 격차로 비중 (1장)
       Yes                ├─ 25% 현금화 리밸런싱 (2-2)
        │                 └─ 과열 시 20% 선제 현금화 (2-3)
        ▼
   [위기 진입]  (마삼 카운트 +1, 달력 월 기준)
   현 손익?  ── +수익 ─► 말뚝 제외 매도
        │     ── -손실 ─► 전량 매도 후 말뚝 재진입
        ▼
   금리 환경 분기
   ├ 비제로금리: -5% 그리드 / ~50% 말뚝 / TLT
   └ 제로금리:   -2.5% 그리드 / ~25% 말뚝 / IAU
        │
   같은 달력 월 -3% 4회 ? ──Yes──► [공황] 말뚝 중단 → 전량 매도 → 현금 대기
        │No                              │
        │                               재진입 트리거(아래) 충족 시 올인
        │                               → 최고점 경신까지 완전 홀드(6-4)
        ▼
   재진입/올인 트리거 (5장) 하나라도 충족 ?
   ├ ①1개월+1일 무마삼 / ②8거래일 상승 / ③1등주 전고 / ④나스닥 전고

   ├ ⑤ 2구간 V자 → 잔여 현금 올인 → (평시)말뚝 유지 (6장)
   │       └ 3구간 재하락 → V자 물량 매도 → 말뚝 재진입
   └ 고점 대비 6구간 하락(비제로 -30% / 제로 -15%) → 보유 현금 전량 긴급 올인 (6-3)
```

| 약점 | 레짐 변화 대응력 / 특정 숫자(8, -3%, -30%) 과최적화 가능성 / 일부 지표 정의 모호 |
| 결론 | "감정 통제용 룰북으로 우수, 수익 극대화 시스템으로는 보완 여지" |

## Z-B. 신호 정의 정교화

- **B-1 마삼 동적 기준(옵션)**: 고정 -3% 유지하되, 검증 후 ATR/VIX 연동 동적 기준 선택 적용.
  - VIX<15 → -2%도 마삼급 / 15~25 → -3% 유지 / >25 → -4~5%로 상향 / 또는 일간 하락 > 1.5×ATR(14).
- **B-2 "8거래일" 검증(옵션)**: 5~10일 범위 백테스트로 8의 유효성 재확인. 보조 신호로 50일선 상향돌파 + 거래량.
- **B-3 1등주 정의(옵션)**: 시총 1위 유지하되, 리더 교체 의심 시 시총+모멘텀(6~12M)+거래대금 3요소 중 2개 교차 확인. 교체는 **평시에만**.
- **B-4 트리거 신뢰도 등급(충돌 없음, 우선 적용 가능)**: ①약 / ②⑤중 / ③④강. 실행 규칙은 원본(단일 신호=올인) 유지, 표시용.

## Z-C. 리스크 관리 강화 (※ 원본 변경 → 백테스트 필수)

- **C-1 긴급 올인 분할(옵션)**: 6구간 하락 시 전량 올인 대신, 4·5·6구간(예: 비제로 -20/-25/-30%) 구간 분할 투입. (원본 "전량 올인" 변경)
- **C-2 올인 구조 완화(옵션)**: 단일 신호=전량 대신 Fixed Fraction(25~50%), 복수 신호 중첩 시 100%.
- **C-3 V자 진입 강화(옵션)**: 2구간 + 거래량 증가 + 5/20일선 돌파 동시 확인. (진입 지연 위험)
- **C-4 손절(옵션, ⚠️ 원본 6장과 정면 충돌)**: V자 진입분 -3~5% 손절은 "올인 후 말뚝 유지" 원칙과 반대. 둘 중 하나만 채택.
- **C-5 공황 단계 청산(옵션)**: 마삼 2회 30%↓ / 3회 60%↓ / 4회 전량. (원본 "4회 일괄 전량" 변경) + 20일선 하회 동시확인 필터.

## Z-D. 레짐 필터

- **D-1**: V자 올인 후 200일선 장기 하향 지속 시 비중 상한 보수적 점검 / 6개월+ 횡보 시 "횡보장 모드". (원본 6장과 상충 가능 → 우선순위 사전 결정)
- **D-2 구조 필터**: 금리(인하/인상)·유동성(완화/긴축)·경기(확장/침체) — 비우호 다수면 올인 비중 한 단계↓.
- **D-2-1 심리·변동성 데이터**:
  - **VIX**(객관·트리거 연동 가능): <15 안정 / 15~25 보통 / 25~35 불안 / >35 공황. 데이터 `^VIX`.
  - **Fear & Greed**(합성·후행, **단독 트리거 금지** 확인용): 0~25 극공포(가중) … 75~100 극탐욕(익절 점검). CNN 비공식 API(`production.dataviz.cnn.io`, 캐싱·예외처리 필수).
- **D-3 QQQ 재정의**: "헤지"가 아니라 **분산/베타 감소**. 진짜 방어는 현금 확대(가장 확실) 또는 인버스ETF(단기·소규모). 갈아타기 보조조건: `1등주/QQQ` 비율 20일 저점 이탈(상대 약세) 시에만 실행.
